# Unity 2D Platformer Controller (FSM)

这是一个基于有限状态机 (FSM) 架构的 Unity 2D 平台动作游戏控制器模板。代码结构干净解耦，专注于提供扎实、精准的平台跳跃基础手感。

## 核心特性

* **FSM 状态机架构：** 输入层与物理逻辑严格分离，添加新状态（如冲刺、受击、下蹲）极其方便，不影响底层逻辑。
* **手感优化：** 内置土狼时间 (Coyote Time) 与跳跃缓冲 (Jump Buffering)，大幅提升操作容错率。
* **物理稳定：** 包含下落重力加速、防粘墙（0摩擦力材质处理）以及精准的 BoxCollider Trigger 底部地面检测。
* **输入优化：** 支持后按键优先逻辑，左右移动切换无卡顿。

### [v1.0] - 初始版本
* **架构底座：** 建立有限状态机 (FSM) 核心，实现输入监听 (`PlayerInput`) 与物理逻辑的完全解耦。
* **基础状态：** 完成站立 (Idle)、跑动 (Run)、空中 (Air) 三大基础状态的平滑流转。
* **手感机制：** 实装“土狼时间” (Coyote Time) 与“跳跃缓冲” (Jump Buffering)；加入后按键优先逻辑，消除左右转向卡顿。
* **物理优化：** 引入下落重力倍率 (Fall Multiplier) 提升落地扎实感；采用 0 摩擦材质配合 Trigger 横条检测，彻底解决“墙壁粘连”与“边缘判定丢失”等底层痛点。
